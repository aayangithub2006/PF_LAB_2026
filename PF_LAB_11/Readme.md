The main objective of Lab 11 is to master the implementation of recursion and structured data types in C programming to solve real-world problems. Specifically, the recursion tasks (Questions 1–3) focus on identifying base cases and recursive steps to calculate exponential growth in power consumption, perform character audits in strings, and find distinct paths in a maze.Simultaneously, the structure tasks (Questions 4–6) aim to teach students how to organize complex data into custom entities—including nested structures—to manage hospital registries, university catalogs, and e-commerce orders.By tracing the recursive call stack for power calculations, such as determining that a 4-floor building consumes 1500 kWh through the sum of $800 + 400 + 200 + 100$, students gain a practical understanding of how memory and function logic handle hierarchical computations.
