In this lab, I will develop my proficiency in C by mastering three core programming pillars: string manipulation, multi-dimensional data structures, and file management. I will learn to use <string.h> library functions like strlen(), strcmp(), and strchr() to validate user input and extract specific data segments. By working with 2D character arrays, I will gain the ability to store and search through lists of strings, such as student names or unique word sets. Finally, I will implement file handling techniques to ensure data persistence, practicing how to create, read, and append to external text files while managing FILE pointers and handling potential errors gracefully.
