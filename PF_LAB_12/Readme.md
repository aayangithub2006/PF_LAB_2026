The objective of this lab is to understand and apply dynamic memory allocation in C using functions such as `malloc`,
`calloc`, `realloc`, and `free`. It aims to demonstrate how memory can be allocated at runtime based on user requirements,
how initialized memory from `calloc` differs from uninitialized memory in `malloc`, and how existing memory can be safely 
resized using `realloc` without data loss. The lab also focuses on proper memory management practices to avoid issues like
memory leaks and dangling pointers, and helps apply these concepts to real-world applications such as contact management 
systems and sensor data storage.
