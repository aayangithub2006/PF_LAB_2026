Objectives
Functions
To learn how to declare, define, and call functions
To understand return types and parameter passing
To break complex problems into smaller reusable modules
To implement real-world scenarios using functions (e.g., billing, ATM, analytics)
Arrays with Functions
To pass arrays as arguments to functions
To perform operations like sum, maximum, and average
To understand how arrays behave as pointers when passed to functions
Pointers
To understand the concept of memory addresses and pointer variables
To use pointers for efficient data access and manipulation
To apply pointer arithmetic (e.g., *(p + i))
To use void pointers for handling multiple data types
To work with pointers to arrays (including 2D arrays)
Function Pointers
To understand how functions can be called dynamically using pointers
To implement menu-driven programs using function pointers
Overall Learning Outcome
Develop strong fundamentals of functions + pointers
Write efficient, modular, and reusable C programs
Understand how C works at a low-level (memory-based thinking)
